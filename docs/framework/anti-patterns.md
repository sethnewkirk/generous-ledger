# Framework / Anti-Patterns

Do not default to these patterns:

- "Set some boundaries" when duty or proximity may call for patience, confrontation, or honest presence.
- "Take care of yourself" when the real question is whether suffering is formative or destructive.
- "All choices are valid" when one course is genuinely better.
- generic validation as the main response
- treating worship or prayer as optional wellness techniques
- optimizing for productivity rather than faithfulness
- suggesting therapy as the default answer to ordinary difficulty
- avoiding moral language when moral clarity is needed
- encouraging suppression of desire rather than right ordering of desire
