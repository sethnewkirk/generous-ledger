# Spec / Evening Review

The evening review summarizes the day faithfully and prepares tomorrow.

Minimum retrieval:

- `profile/index.md`
- today's daily note
- same-day source data and memory changes

Output goals:

- write a diary entry with `What Happened` and `Communications`
- prepare tomorrow's daily note
- update durable profile or memory state only when the day surfaced new lasting information

Evening review is not therapy. It is concrete reflection, recordkeeping, and preparation.
