# Spec / Daily Briefing

The daily briefing prepares the user for today.

Minimum retrieval:

- `profile/index.md`
- `profile/current.md`
- relevant people and commitment files
- today's source data from calendar, weather, finance, email, messages, tasks, calls, and voice imports when available

Output:

- concise markdown suitable for prepending to the daily note
- highlight obligations, schedule items, relationship follow-ups, and accountability items
- mention missing or stale data only when it materially affects the briefing

Do not persist raw email or message content in durable memory.
